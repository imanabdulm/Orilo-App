# Security Policy & Privacy Statement 🔒

Orilo is built from the ground up as a **100% private, local-first macOS app**.

---

## 🛡️ Privacy Guarantee

- **No Remote Telemetry:** Orilo does not collect, track, or send any user data to external servers.
- **Local Persistence:** All focus sessions, preferences, and recaps are stored locally on your device in `~/Library/Application Support/Orilo/`.
- **Zero Third-Party SDKs:** No analytics, tracking pixel, or telemetry libraries are included.

---

## 🐛 Reporting Security Issues

If you discover a security vulnerability or concern within Orilo, please report it directly via [GitHub Security Advisories](https://github.com/imanabdulm/Orilo-App/security/advisories) or by opening a confidential inquiry. We aim to respond to security reports promptly.
